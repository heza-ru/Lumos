How native glass will be wired in Lumos
1 Plugin in Cargo.toml
Add tauri-plugin-liquid-glass to Cargo.toml. It wraps NSGlassEffectView — Apple's private macOS 26 API — with a safe Tauri command layer. Falls back to NSVisualEffectView on macOS <26.
2 Apply to toolbar window
Call setLiquidGlassEffect({ cornerRadius: 100, variant: "Regular" }) from the toolbar window on startup. The system Metal compositor renders real lensing, specular highlights, and refraction beneath the React content.
3 React renders on top
The webview background is fully transparent — background: transparent. React only renders the icons, separators, and interactive elements. The glass material is entirely handled by macOS, so there are no rough edges, no CSS artifacts.
